## 4. التوصية بالاستخدام (Recommended Uses)

### أُنشئت لـ
- تدريب وتقييم نماذج فصل المتحدثين (Speaker Diarization).
- تدريب وتقييم نماذج التفريغ التلقائي (ASR).
- استخلاص الكيانات والكلمات المفتاحية من النص العربي.
- بناء شبكات معرفة تربط متحدثين وكلمات وأحداث.

### **لا** يُنصح باستخدامها لـ
- التعرّف الصوتي البيومتري في تطبيقات إنفاذ القانون.
- التصنيف القاطع لأي محادثة بأنها «مشبوهة» دون مراجعة بشرية.
- النشر التجاري دون مراجعة قانونية للترخيص (خاصة مقاطع Common Voice).
- التطبيقات التي تتخذ قرارات أوتوماتيكية تؤثر على حياة الناس.

### القيود المعروفة
- **اللهجات:** التغطية غير متوازنة، الفصحى والخليجية قد تكون مُمَثَّلة أكثر من غيرها.
- **الجنس:** قد تكون الأصوات الذكورية أكثر من النسائية في Common Voice.
- **العمر:** غالب البيانات لمتحدثين بالغين، لا تغطية كافية للأطفال أو كبار السن.
- **جودة الصوت:** غالباً نظيفة (ميكروفون مكتبي)، لا تمثّل ضوضاء بيئات حقيقية كاملة.
- **اللهجات النادرة:** المغاربية والسودانية أقل تمثيلاً في Common Voice.

## 7. الاعتبارات الأخلاقية

### هل أُنشئت بموافقة من المتحدثين؟
- مقاطع Common Voice: نعم، المتحدثون متطوعون وافقوا على CC0.
- التسجيلات الخاصة: نعم (اتفاقية مُوقَّعة + حق الانسحاب في أي وقت).

### كيف نتجنّب الأذى؟
- لا تُستخدم البيانات لاتخاذ قرارات أوتوماتيكية.
- النظام يقدّم اقتراحات يراجعها بشر.
- حق المحو: أي متحدث له الحق بطلب حذف بياناته بالكامل.
- شفافية كاملة: نوثّق حدود النظام وتحيّزاته في التقرير النهائي.

### تحيّزات معروفة
- جنس، لهجة، عمر — راجع قسم القيود.
- التوصية: تقييم الأداء المُنفصل لكل فئة في تقرير النتائج.

## 8. إعادة الإنتاج

### كيف أعيد إنتاج المجموعة بالضبط؟
```bash
git clone <repo>
cd call-intelligence-engine
pip install -r requirements.txt

# 1. تحميل
python scripts/download_common_voice.py --max-clips <N>

# 2. تقسيم
python scripts/prepare_dataset.py --strategy speaker

# 3. تحقق
python scripts/validate_dataset.py
```

### السجلّات (manifests)
- `data/raw/common_voice/manifest.json` — كل المقاطع المحمّلة.
- `data/train/manifest.json` — تقسيم التدريب.
- `data/val/manifest.json` — تقسيم التحقق.
- `data/test/manifest.json` — تقسيم الاختبار النهائي.
