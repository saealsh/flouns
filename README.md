# Call Intelligence Engine — الديمو المباشر

ديمو كامل يعمل في المتصفح **بدون أي خادم أو تثبيت**.

## التشغيل

1. فك ضغط الملف
2. افتح **`index.html`** بالضغط عليه مرتين

سيعمل الموقع مباشرة في متصفحك.

## ماذا يعمل في الديمو

- ✅ كل الصفحات الست (لوحة تحليل، مكالمات، رفع، شبكة معرفة، متحدثون، تقارير)
- ✅ البحث والفلاتر على جميع القوائم
- ✅ عرض تفاصيل أي مكالمة / متحدث / تقرير
- ✅ شبكة المعرفة التفاعلية
- ✅ **رفع ملفات حقيقية** — جرّب رفع ملف `.txt` أو `.srt` وسيُحلَّل في المتصفح فوراً
- ✅ استخلاص الكيانات والكلمات المشفّرة من النصوص المرفوعة
- ✅ البيانات تُحفظ في المتصفح (تبقى حتى لو أغلقت الصفحة)

## جرّب هذا

**١. ارفع ملف نصي** — أنشئ ملف `test.txt` واكتب فيه:

```
أحمد: السلام عليكم، الشحنة جاهزة للإرسال يوم الخميس.
خالد: وعليكم السلام. تأكد من العربة الجديدة قبل الإرسال.
أحمد: طيب، هل نثبت الاجتماع قبل التسليم؟
خالد: نعم، الساعة التاسعة صباح الاثنين عند النقطة المعتادة.
```

اذهب لصفحة "رفع ملف" → اسحبه → اضغط "ابدأ التحليل" → سترى المحرك يستخرج:
- أسماء المتحدثين تلقائياً
- الكلمات المشفّرة ("العربة الجديدة"، "النقطة المعتادة")
- الأحداث ("تسليم"، "اجتماع")
- الأزمنة ("يوم الخميس"، "الاثنين")

**٢. الصيغ المدعومة:** `.txt`, `.srt`, `.vtt`, `.json`, `.csv` (نص) + أي ملف صوتي (توليد تفريغ تجريبي).

## إعادة التعيين

اضغط على أيقونة `↻` في أعلى الصفحة لمسح كل البيانات المُضافة والعودة للبيانات الأصلية.

## الفرق عن النسخة الكاملة

هذا الديمو لا يحتاج Python — كل المنطق يعمل بـ JavaScript في المتصفح. النسخة الكاملة مع Backend حقيقي (FastAPI + SQLite + دعم ربط Whisper/Claude) موجودة في ملف `call-intelligence-full-v2.zip`.
